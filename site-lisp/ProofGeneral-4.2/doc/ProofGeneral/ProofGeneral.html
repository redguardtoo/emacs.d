Proof General

This file documents version 4.2pre of Proof General, a generic Emacs interface for proof assistants.

Proof General 4.2pre has been tested with GNU Emacs 23.3 on Linux. It is supplied ready to use for the proof assistants LEGO, Coq, Isabelle, and PhoX. Experimental support is provided for several other provers.
